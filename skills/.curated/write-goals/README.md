# Write Goals

Draft, critique, or rewrite Codex `/goal` objectives for long-running work.

Use this skill when a rough task needs to become a durable goal with a clear objective, baseline, scope boundaries, verification, stop condition, and pause/block criteria. It keeps simple goals lightweight while adding stronger verifier, anti-cheating, and review-pressure rules for risky or long-running goals.

The skill is instruction-only. It does not create files, start goals, spawn subagents, or set token budgets unless the user explicitly asks for those actions.

## Install

```bash
$skill-installer install https://github.com/vanderhaka/skills/tree/main/skills/.curated/write-goals
```

Restart Codex after installing new skills.
