# dashboard-ui-ux

`dashboard-ui-ux` guides Codex through building, critiquing, and refining dense SaaS dashboards, admin panels, analytics views, CRM consoles, tables, KPI rows, chart surfaces, filters, and contextual actions.

Use it when a dashboard needs to feel operational and production-ready rather than like a landing page. It emphasizes compact typography, real desktop sidebar collapse, card/background contrast, useful table tooling, hoverable chart details, realistic mock data, and browser proof.

Install:

```bash
$skill-installer install https://github.com/vanderhaka/skills/tree/main/skills/.curated/dashboard-ui-ux
```

Restart Codex after installing new skills.
